# My Godot Game

## How to test the game

1. Download the project files.  
2. Open Godot Engine.  
3. Click on **Import**.  

<img src="/src/image.png" width="500" />

4. Select the project folder and choose the `project.godot` file.  
5. Click **Import & Edit**.  
6. Press **F5** or click the **Play** button to run the game.  

<img src="/src/play.png" width="500" />
